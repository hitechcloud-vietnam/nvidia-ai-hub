# GraphRAG Workstation Projects

Each subfolder in this directory is treated as one GraphRAG dataset workspace.

Recommended structure per dataset:

- `input/` for raw `.txt` or `.csv` source files
- `settings.yaml` generated by `graphrag init`
- `.env` for GraphRAG model credentials and endpoint overrides
- `prompts/` generated by `graphrag init`
- `output/` generated by `graphrag index`
- `metadata.json` for optional local display metadata used by this workstation

The included Streamlit workbench can initialize a new dataset folder, create a starter sample file, run indexing, and compare query modes after the dataset has been indexed.
