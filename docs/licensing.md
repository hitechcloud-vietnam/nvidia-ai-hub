## What Counts as Non-Commercial Use

Examples typically allowed under the repository license:

- personal learning
- private experimentation
- academic work
- educational lab use
- internal research
- internal testing and evaluation
- internal non-commercial prototyping with no revenue generation or client delivery

These examples are illustrative, not exhaustive.

## What Counts as Commercial Use

Examples that require separate written permission:

- paid consulting or implementation services
- deploying the project for client work
- bundling the project in a paid offering
- hosting the project as part of a paid service or SaaS
- reselling, sublicensing, or monetizing the software
- using the software in revenue-generating operations
- including the software in a commercial managed service, subscription, or marketplace listing

## Practical Boundary Guide

### Usually allowed

- a developer evaluating the project locally
- a school using it in a classroom or lab
- an internal engineering team testing compatibility without revenue generation
- a researcher benchmarking or studying the project in a non-commercial context

### Usually not allowed without separate permission

- an agency deploying it for a client
- a company integrating it into a paid platform
- a consultant using it in billable delivery work
- a provider hosting it for customers
- redistribution as part of a commercial bundle or product

## Internal Company Use

Internal company use is allowed only when it remains strictly non-commercial.

If the use supports client delivery, paid services, production monetization, or commercial advantage, treat it as commercial use and obtain separate written permission.

## Derivative Works

You may modify the repository and create derivative works only within the non-commercial scope of the license unless separate written commercial rights are granted.

Creating a derivative work does not remove the commercial restrictions.

## Redistribution

Permitted redistribution must:

- retain the license and copyright notice
- clearly state that the software is licensed for non-commercial use only
